# iicocece DEAP: 1,000 Usage Analytics and Decision-Intelligence Metrics

Use this catalogue as the admin analytics blueprint for DEAP. Each metric can be filtered by date range, department, user, test, question bank, difficulty, topic, supervisor, and role where applicable.

## Source-Informed Foundation
- LMS and learning analytics: logons, completion, engagement, time spent, learner risk, cohorts, dashboards, and report scheduling.
- Assessment item analysis: item difficulty, discrimination, point-biserial logic, response frequency, distractor performance, and reliability.
- Product analytics: active users, session duration, retention, feature adoption, funnels, and user behaviour.
- Compliance training analytics: scores, time spent, retries, confidence, role/department segmentation, and audit-ready reporting.
