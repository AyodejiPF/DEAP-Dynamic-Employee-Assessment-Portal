# Digital Asset Production

Use this skill to manage visual and document assets like a production pipeline.

## Workflow

1. Identify source assets and destination folder.
2. Preserve originals and write outputs to a separate location.
3. Create an asset register or manifest.
4. Define naming, folder structure, format, resolution, theme, and version rules.
5. Run a small pilot when quality or cost is uncertain.
6. Create review sheets or previews for fast approval.
7. Flag assets that need text, brand, quality, or human review.
8. Record run notes, failures, limits, and next steps.

## Quality Gates

Check readability, brand match, visual consistency, file completeness, duplicate files, missing outputs, and whether the asset can be reused later.

For generated images, record model limitations and avoid overwriting source artwork.

## Output Format

Report source, destination, manifest, count, failures, review path, approval status, and next production step.
