# AI Workbench

A portable hub on this Desktop that holds the shared building blocks for every AI coding
tool you use.

## Contents

- Skills            : reusable skills, each as a subfolder with a SKILL.md.
- Global Settings   : shared preferences and configuration every tool should honour.
- IDE_REFERENCE_MESSAGE.md : the block to paste into any tool, plus how to wire each tool
                              to this hub for real.

## Idea

One place to define skills and settings once, and have Claude Code, Claude CLI, Cowork,
Antigravity, Codex and others reference the same source, so you are not repeating yourself
across tools.
